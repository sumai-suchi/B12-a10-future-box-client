# B12-A10-Future-Box-client

# Description:
       
       This is a Learning website. Where we can take different type of courses and 
       also do different type of Courses for skill development.This react website is connected with mongodb database and node.js express.js backed.

       # Pages:
            
            It has in its  home page  popular courses, Instructor and why us section .
            In AllCourses page it has a heading ,filter option category wise and all the courses in cards.In dashboard page it have some links for different pages like my enrolled courses , add courses with a form and My added course that user added.In this we have delete update and view details functionality.

       #Authentication:
                  
            We use firebase for authentication .In the website we have login and register page.    

          there is a footer section with Logo of the website and social logins.
